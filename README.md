# Comandos 
$ cd /c/wamp/www 
- Coloca o ponteiro na pasta c:/wamp/www
$ mkdir nome_pasta 
- Cria uma pasta com o nome nome_pasta no /local/do/ponteiro

$ git init 
- Inicia um repositório Git. O Repositório é onde ficam os arquivos do projeto. Você sabe que está no diretório de um repositório quando aparece o nome do Branch (master) no Git Bash
$ git clone URL_ou_local_da_pasta nome_pasta
- Cria uma cópia idêntica de um repositório Git na pasta nome_pasta
